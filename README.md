# Titanic Survival Prediction Using Machine Learning

## Objective
This project uses Machine Learning algorithms to predict Titanic passenger survival based on passenger information.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## Machine Learning Algorithms
- Decision Tree Classifier
- Random Forest Classifier

## Features Used
- Passenger Class
- Gender
- Age
- Siblings/Spouses
- Parents/Children
- Fare
- Embarked Port

## Tasks Performed
- Data cleaning
- Missing value handling
- Encoding categorical data
- Train-test split
- Model training
- Model evaluation
- Visualization

## Evaluation Metrics
- Accuracy Score
- Confusion Matrix
- Classification Report
- ROC Curve

## Conclusion
Random Forest achieved better prediction performance compared to Decision Tree. The project demonstrates the complete supervised machine learning workflow.
